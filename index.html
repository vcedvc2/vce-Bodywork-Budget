Connect to Google Sheets

Your repair data auto-saves to a Google Sheet you own. Set up once in ~3 minutes.

  1. Go to script.google.com → New Project
  2. Delete all code. Download VCE_Backend.gs → paste it in → Save
  3. Click Deploy → New Deployment
  4. Type: Web App · Execute as: Me · Access: Anyone
  5. Click Deploy → approve permissions → copy the Web App URL
  6. Paste the URL below and click Connect

Or skip for now — data saves locally in your browser
